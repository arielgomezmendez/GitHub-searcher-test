Instrucciones:
1. Crea un repositorio público en GitHub para el proyecto del examen.
2. Clona el repositorio en tu máquina local.
3. Crea un nuevo componente de React llamado `GitHubUser` en tu proyecto.
4. El componente `GitHubUser` debe tener las siguientes funcionalidades:
   - Permitir al usuario ingresar el nombre de usuario de GitHub en un campo de texto y mostrar la información correspondiente al presionar un botón.
   - Mostrar el nombre de usuario, la foto de perfil y la biografía del usuario.
   - Mostrar el número de seguidores y el número de repositorios públicos del usuario.
   - Mostrar una lista de los repositorios más recientes del usuario, incluyendo el nombre y la descripción de cada repositorio.
5. Utiliza la API pública de GitHub (https://docs.github.com/en/rest) para obtener los datos necesarios.
6. Realiza commits periódicos a medida que desarrollas la funcionalidad en tu proyecto local.
7. Sube tus cambios al repositorio remoto en GitHub.
8. Envía el enlace del repositorio público de GitHub como parte de la entrega del exámen.

Recuerda que el objetivo de esta prueba es evaluar tanto tus habilidades en el desarrollo con React y el uso de una API pública, como tú conocimiento y manejo de Git para el control de versiones y colaboración en un repositorio remoto.
